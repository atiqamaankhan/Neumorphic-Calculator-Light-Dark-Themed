# Neumorphic Calculator (Light/Dark Mode)

A modern **web-based calculator** built using **HTML, CSS, and JavaScript** with a beautiful **Neumorphic UI design** and a **light/dark theme toggle**.

This project demonstrates **front-end UI design, DOM manipulation, and JavaScript logic** while creating a visually appealing calculator interface.

---

## Features

-  Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
-  Decimal number support
-  Clear (`C`) button to reset calculations
-  Modern **Neumorphic UI design**
-  **Light / Dark mode toggle**
-  Interactive and responsive buttons
-  Clean and minimal user interface

---

##  Technologies Used

- **HTML5** – Structure of the calculator  
- **CSS3** – Styling, neumorphic effects, and layout  
- **JavaScript (Vanilla JS)** – Calculator logic and theme switching
